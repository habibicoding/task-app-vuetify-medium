## Project setup

```
# pnpm
pnpm install
```

### Compiles and hot-reloads for development

```
# pnpm
pnpm dev
```

### Compiles and minifies for production

```
# pnpm
pnpm build
```

### Lints and fixes files

```
# pnpm
pnpm lint
```

### Run tests

```
# pnpm
pnpm test:unit
```

### Run Tests with coverage

```
# pnpm
pnpm test:coverage
```

See [Configuration Reference](https://vitejs.dev/config/).
